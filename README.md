**VideoOptimizer** is a lightweight desktop tool that allows users to compress video files and significantly reduce their size while preserving visual quality. The app leverages the power of **FFmpeg** to perform efficient transcoding and bitrate optimization.

## Key Features:
- **FFmpeg-powered processing**: Uses the FFmpeg engine internally for reliable and efficient compression;
- **Custom compression level**: Choose the desired compression strength to balance file size and visual quality;
- **Wide format support**: Works with almost all popular video formats;
- **Simple and clean UI**: Minimalistic interface designed for quick use without unnecessary options.

## Technologies and Tools:
- **Engine**: Unity 6000.0.24f1 (LTS);
- **Programming Language**: C#;
- **Architecture**: Application of SOLID principles;
- **Design Patterns Used**: Observer, Dependency Injection;
- **Additional Tools**: DOTween, Zenject, UniTask, FFmpeg, StandaloneFileBrowser, Borderless.
